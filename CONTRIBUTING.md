# Contributing

Interested in improving this project? Contributions are welcome!

## How to Contribute

1. **Report Issues**  
   Found a bug or have a suggestion? Please open an issue on GitHub with clear steps to reproduce or a detailed description of your idea.

2. **Suggest Enhancements**  
   Have ideas for new features, alternative visualisations, or performance improvements? Share them in an issue so we can discuss the best approach.

3. **Submit Pull Requests**  
   - Fork the repository  
   - Create a feature branch  
   - Make your changes  
   - Submit a pull request with a clear description of what you changed and why  

## Commit Message Guidelines

Please follow the conventional commit style for clear, searchable history:

- `feat:` New feature  
- `fix:` Bug fix  
- `docs:` Documentation changes  
- `style:` Code formatting (no behaviour change)  
- `refactor:` Code restructuring without changing behaviour  
- `perf:` Performance improvements  
- `test:` Adding or updating tests  

For more details, see: https://www.conventionalcommits.org/
